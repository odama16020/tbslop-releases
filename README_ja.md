# tbslop 説明書

> 📥 **ゲームのダウンロードは、ページ右側の「Releases」からどうぞ。** 最新版の Linux / Windows バイナリを配布しています。

tbslop はテキストベースインターフェースのミニマルなターンベース戦術ゲームです。
味方ユニットを配置し、移動、攻撃、スキル、再行動を組み合わせて、制限ターン以内にマップの目標達成を目指します。

## 操作説明

### キーボード

| 入力 | 操作 |
| --- | --- |
| WASD / 矢印キー | カーソル移動。マップカーソルは Shift を押しながらで高速化 |
| Z / Space | 決定 |
| E / Enter | 決定、メニュー遷移 |
| X / Esc | キャンセル |
| R | 危険範囲の表示切替 |
| L | ログ表示 |
| H | ユニット情報/ヘルプ |
| U | Undoメニュー |
| Ctrl+U | クイックUndo |
| Ctrl+S / Ctrl+L | セーブ / ロード |
| O | オプション |
| Ctrl+C | 終了確認 |
| Ctrl+Q | ゲーム中にステージ選択へ戻る |
| 0-9 | 対応する味方ユニットへジャンプ |

ステージ選択では `L` でロード画面を開きます。ゲーム中にユニットのいないマスで決定するとメニューを開きます。

### XInput

| 入力 | 操作 |
| --- | --- |
| 十字キー / 左スティック | カーソル移動 |
| A | 決定 |
| B | キャンセル |
| X | Undoメニュー |
| Y | ユニット情報/ヘルプ |
| LB / RB | 前/次の生存味方ユニットへカーソル移動。セーブ/ロード画面ではタブ切替 |
| LT | ログ表示 |
| RT | 危険範囲の表示切替 |
| START | ステージ選択ではロード。配置中は開始。プレイヤーフェーズではフェーズメニュー |
| SELECT / Back | オプション |
| L3 | ボスへジャンプ |

敵フェーズを手動進行にしている場合、A または B で次の敵行動へ進み、START でスキップ確認を開きます。

### 基本

- 通常マップの配置フェーズでは、出撃ユニットを選び、`$` で示された配置候補マスに配置します。最大配置数まで配置したら開始できます。チュートリアルではユニット固定配置です。
- プレイヤーフェーズでは、味方ユニットを選択して移動先を決め、その後に攻撃、スキル、再行動、待機などのコマンドを選びます。
- エネミーフェーズでは敵AIが行動します。手動進行設定では、敵行動を1つずつ確認できます。

## ゲームルール

### 基本

ターン制限以内にマップの目標を達成すれば勝利です。多くのマップでは「敵全滅」または「ボス撃破」が目標です。
味方ユニットが全滅した場合、または制限ターンの敵フェーズ終了時点で勝利していない場合は敗北します。

フェーズは `配置 -> プレイヤーフェーズ -> エネミーフェーズ -> プレイヤーフェーズ...` の順に進みます。プレイヤーフェーズ中に全味方が行動済みになっても、デフォルトでは自動ではターン終了しないため、必要ならフェーズメニューからターン終了を選んでください。 (設定で自動ターンエンド可)

### 移動と地形

移動は上下左右の4方向で、各ユニットは `Mv` の値まで移動できます。

| 地形 | 表示 | 基本ルール |
| --- | --- | --- |
| 床 | `.` | 通行可能 |
| 壁 | `#` | 通行不可 |


| 表示 | 意味 |
| --- | --- |
| HP | 0 になると撃破されます |
| Pw | 攻撃力です。`x2` などが付く場合、その回数だけ連続攻撃します |
| Ht | 命中です。攻撃側の Ht が防御側の Ev 以上の場合に命中します |
| Ev | 回避です。相手の Ht より高いなら攻撃を回避します |
| Sp | 速度です。相手より高い側が追撃します |
| Df | 物理防御です。物理攻撃のダメージを軽減します |
| Rs | 魔法防御です。魔法攻撃のダメージを軽減します |
| Mv | 移動力です |

### コマンド

| コマンド | 説明 |
| --- | --- |
| 攻撃 | 通常攻撃します。射程はユニットごとに異なります |
| スキル | AP を消費してコマンドスキルを使います。範囲攻撃や補助効果を持つものがあります |
| 再行動 | AP を消費してそのユニットを再行動可能にします |
| 待機 | 行動を確定して待機します |
| 再移動 | 一部効果で可能になる行動後移動です。行動後、残り移動力の範囲で移動できます |

パッシブスキルは条件を満たすと自動で働きます。コマンドスキルはユニットごとのスキルコマンドから使用します。

### AP

APはプレイヤーユニット全員で共有するリソースです。スキルと再行動で消費します。
マップ開始時に一定量を持つほか、敵撃破で1獲得できます。

### 鼓舞 (Boost)

プレイヤーフェーズ中に敵を撃破すると、撃破したユニットが持つ鼓舞効果が味方全体に自動付与されます。
鼓舞の内容はユニットごとに異なります。同時に維持できる数には上限があり、上限を超えると古い鼓舞から失効します。

### 再行動 (Extra)

APを消費して、ユニットをもう一度行動可能にします。すでに待機状態でもAPがあれば使用可能です。
同じターン中に複数回使えますが、同じユニットに使うほど必要 AP が増えます。ターンが変わると使用回数はリセットされます。

### セーブ、ロード、Undo

マップ中は手動セーブとロードができます。オートセーブ設定を有効にすると、プレイヤーフェーズ開始時またはユニット行動ごとに自動保存します。
Undoはプレイヤーフェーズ中に使えます。Undoメニューでは戻る地点を選べ、クイックUndoは直前の手順へ戻ります。保存データにはUndo履歴も含まれます。
